The NWS deployer app is able to deploy
your git repo to your LAMP environment.

Automatic Steps:
- Running checkups
- Create new "TEMP" folder four building
- create new "destination" folder with specific name and date encoded
- Pull Your PUBLIC (later private) repo into the "TEMP" folder
- checkout branch
- running type processor
    - Vue:
        - npm install
        - npm run build
        - copy dist folder to "destination"
        - remove "TEMP" dir
        - Update Apache Site Settings

    - Laravel:
        - composer install
        - npm install
        - npm run build
